# ⚡ Meus Comandos Favoritos

Aqui estão os comandos que mais utilizei na aula de Terminal e o que aprendi sobre a aplicação prática de cada um deles:

- `cd` *(Change Directory)*: Usado para navegar entre as pastas do sistema. Durante o exercício, utilizei comandos como `cd Desktop` e `cd missao-terminal` para acessar os locais corretos.
- `dir` *(Directory)*: Comando essencial no Windows para listar todos os arquivos e subpastas dentro do diretório atual. Usei para verificar se a pasta da missão foi criada corretamente e para exibir as informações na tela.
- `mkdir` *(Make Directory)*: Utilizado para criar novos diretórios (pastas) a partir do terminal. Foi o comando que apliquei para gerar a pasta base do desafio (`mkdir missao-terminal`).
- `type nul >` ou `echo`: Aprendi que estes são os recursos no Windows para criar um arquivo de texto diretamente pela linha de comando (ex: `echo pronto > sucesso.txt`), agilizando o processo sem precisar abrir uma interface gráfica.
- `cls` *(Clear Screen)*: Um comando muito útil para limpar a tela do CMD. Ele apaga o histórico visual de comandos anteriores, ajudando a manter o ambiente organizado para tirar prints mais limpos.
